# 💜 Lavender - E-commerce

🌐 Em breve online (deploy em andamento)

---

## Sobre o projeto
Lavender é um e-commerce desenvolvido com React, focado em uma experiência moderna e intuitiva.  

O usuário pode navegar pelos produtos livremente e realizar ações como adicionar ao carrinho e favoritos após autenticação.

## Funcionalidades

- Navegação de produtos sem login
- Autenticação de usuário
- Adicionar ao carrinho
- Sistema de favoritos
- Página de detalhes do produto
- Recomendação de produtos

---

##  Tecnologias utilizadas

- React
- Styled-components
- Supabase
- JavaScript
- HTML5 / CSS3

---

##  Backend

Utilização do Supabase para:
- Autenticação
- Banco de dados
- Storage de imagens
